## Getting Started

### Prerequisites

- Java 17
- Maven
- Docker

### Building

```bash
mvn clean install
```

### Running the application

```bash
mvn spring-boot:run
```

### Building the Docker image

```bash
docker build -t nexosalud/nexo-history-template:latest .
```
